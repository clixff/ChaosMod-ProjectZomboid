[h1]Project Zomboid Chaos Mod[/h1]

[b]A single-player chaos mod for Project Zomboid B42 with Twitch integration.[/b]

This mod triggers a random effect every 45 seconds. There are currently [b]230+ effects[/b], and they can be helpful, dangerous, weird, or completely random.

[h2]Game Version[/h2]

This mod requires the [b]B42 version[/b] of Project Zomboid.
To use it, enable the [b]unstable branch[/b] in the game's Steam properties.

Latest tested version: [b]42.17.0[/b]

[h2]Examples of effects[/h2]

[list]
[*]Spawn Zombie Nearby
[*]Give Random Tool
[*]Enable Rain
[*]Spawn Vehicle Nearby
[*]Turn Power Off
[/list]

[h2]Twitch Support[/h2]

Viewers can vote for the next effect, and Twitch nicknames can appear above zombies.
You can read more about Twitch integration on the mod's GitHub page.

[h2]Languages[/h2]

The mod supports [b]9 languages[/b].
You can read more about supported languages on the mod's GitHub page.

[h2]Multiplayer[/h2]

This mod does [b]not[/b] support multiplayer.
It is designed for single-player only.

You can read more about changing the mod's settings on the mod's GitHub page.
