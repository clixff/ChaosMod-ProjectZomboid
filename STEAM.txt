[b]A single-player Chaos Mod v1.1.1 for Project Zomboid B42 with Twitch integration.[/b]

This mod triggers a random effect every 45 seconds. There are currently [b]310+ effects[/b], and they can be helpful, dangerous, weird, or completely random.

[b]Twitch integration[/b] is optional and allows your viewers to vote for the next effect. You can play without Twitch integration.

Viewers can vote for the next effect, and Twitch nicknames can appear above zombies.
To enable Twitch support, you need to install StreamerApp from [url=https://github.com/clixff/ChaosMod-ProjectZomboid]GitHub Page[/url] of the mod.

You can read more about Twitch integration on the mod's GitHub page.

[img]https://raw.githubusercontent.com/clixff/ChaosMod-ProjectZomboid/refs/heads/master/assets/steam_v2/steam_preview_06.png[/img]

You can edit the mod settings (effect interval time, vote start time, effect settings, language) in mod settings. You can open mod settings window by pressing button with gear icon in game (bottom-left corner). Also settings are available in StreamerApp dashboard.

[img]https://raw.githubusercontent.com/clixff/ChaosMod-ProjectZomboid/refs/heads/master/assets/steam_v2/steam_preview_05.png[/img]

[b]Q: Mod does not work. What should I do?[/b]
A: Make sure you are using the unstable branch of Project Zomboid on Steam.



[b]Q: How can to disable some effect or change effect settings?[/b]
A: You can edit settings of every effect in the mod settings windows (bottom-left corner)

[b]Q: I only want to display Twitch nicknames above zombies, without random effects and without voting. How can I do this?[/b]
A: In StreamerApp dashboard set these settings:
[list]
[*]Effects interval enabled: [b]false[/b]
[*]Streamer mode enabled: [b]true[/b]
[*]Voting enabled: [b]false[/b]
[*]Donations enabled: [b]false[/b]
[*]Use zombie nicknames: [b]true[/b]
[/list]
